# T1onTour

A module federated Micro FrontEnd (MFE) application that uses a crowd-sourced collection data points & map functionality to display insulin pick-up locations around the world.

MFE breakdown:
- host: overall frontend to be deployed for use
  - header logo
  - header title
  - nav bar
- submit/search: remote app that deals with form filling
- map: the actual map functionality to explore locations
